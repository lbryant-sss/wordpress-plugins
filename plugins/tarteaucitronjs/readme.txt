== Description ==

Get a compliant and accessible cookie banner

Automatic service detection, colors switcher, texts customization.

Free for 7 days, licence required after the trial.

== Installation ==

= WordPress Admin Method =
1. Go to you administration area in WordPress `Plugins > Add`
2. Look for `tarteaucitron.io` (use search form)
3. Click on Install and activate the plugin
4. Find the settings page through `Settings > tarteaucitron.io`

= FTP Method =
1. Upload the complete `tarteaucitronjs` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Find the settings page through `Settings > tarteaucitron.io`
